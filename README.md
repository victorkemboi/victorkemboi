# Hi, I'm Victor Kiprop

### Senior Software Engineer | Fintech Platforms, Mobile Systems & Agentic Engineering

I'm a senior software engineer based in Nairobi, focused on fintech platforms, payment workflows, backend systems, and production-grade mobile experiences.

My background started in deep Android and mobile platform engineering, then expanded into backend and product-system ownership: API-backed products, merchant workflows, payment integrations, offline-first systems, CI/CD quality gates, observability, release safety, and agentic engineering workflows.

I care about building software that works in real operating environments: unreliable networks, payment edge cases, field teams, merchant operations, production incidents, and users who need the product to be fast, stable, and dependable.
